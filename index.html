Scription

a tiny browser-based programming language
Live REPL →

Scription is a small language you write in the browser and Run with a button. It has variables, conditionals, sprites you can move around a stage, on-screen buttons, key handlers, and a screen-flash command — enough to prototype tiny games and animations in a few lines.

Every command in the language is listed below. Use the search box to filter, or jump to a category.

Example
var(hello)
(hello) = "world"
text(hello)

make sprite (circle'blue) name = (bob) size = ("50'50'px")
show (bob)
d pressed = move (bob) 5px right repeat until hold = false

Output

text

Prints a value to the Output pane. Accepts a quoted string or a variable name.

Syntax
text("string")
text(varname)
Example
text("hello world")

var(name)
(name) = "scription"
text(name)

screen text

Places text on the Stage at the given pixel coordinates.

Syntax
screen text("X'Y'px")("string")
screen text("X'Y'px")(varname)
Example
screen text("200'450'px")("hello")

Variables

var

Declares a variable. Must start with a letter or underscore.

Syntax
var(name)
Example
var(score)

(name) = value

Assigns a value to a previously declared variable. The right-hand side can be a quoted string or a bare token.

Syntax
(name) = "value"
(name) = value
Example
var(score)
(score) = "0"
The variable must be declared with var(...) first.

input

Opens a browser prompt and stores what the user types into the variable.

Syntax
input(name)
Example
var(name)
input(name)
text(name)

Control flow

if (name) = value <statement>

Runs the statement when the variable equals the value.

Syntax
if (name) = value <statement>
Example
var(score)
(score) = "10"
if (score) = 10 text("you win")

if (name) ! value <statement>

Runs the statement when the variable does not equal the value.

Syntax
if (name) ! value <statement>
Example
var(state)
(state) = "ready"
if (state) ! done text("keep going")

Sprites

make sprite

Creates a sprite on the Stage. Shape is one of circle, square, triangle, or line. Color can be any CSS color.

Syntax
make sprite (shape'color) name = (name) size = ("W'H'px")
Example
make sprite (circle'blue) name = (bob) size = ("100'100'px")
make sprite (square'red) name = (box) size = ("60'60'px")
make sprite (triangle'green) name = (tri) size = ("80'80'px")
make sprite (line'black) name = (bar) size = ("120'2'px")
For a line, pick a thin height (or width) — it renders as a colored rectangle at the size you give.

show

Makes a sprite visible.

Syntax
show (name)
Example
show (bob)

hide

Hides a sprite.

Syntax
hide (name)
Example
hide (bob)

move

Moves a sprite N pixels in one of four directions. Add repeat until hold = false inside a key handler to keep moving while the key is held.

Syntax
move (name) Npx up
move (name) Npx down
move (name) Npx left
move (name) Npx right
move (name) Npx <dir> repeat until hold = false
Example
move (bob) 5px down
d pressed = move (bob) 5px right repeat until hold = false

move to connected

Teleports the sprite to coordinates stored in its connected variable. The variable's value must be in X'Y'px format.

Syntax
move (name) to connected
Example
var(pos)
(pos) = "100'200'px"
make sprite (circle'red) name = (dot) size = ("20'20'px")
sprite (dot) connect var(pos)
move (dot) to connected

teleport

Moves a sprite or button to absolute coordinates, or to a random position inside the Stage.

Syntax
teleport (name) to (X'Y'px)
teleport (name) to random
Example
teleport (bob) to (300'150'px)
teleport (bob) to random

sprite connect var

Links a sprite to a variable so move … to connected can read coordinates from it.

Syntax
sprite (name) connect var(varname)
Example
sprite (bob) connect var(hello_world)

Buttons & input

button

Creates a clickable button on the Stage. The third group is an identifier you reference from a pressed(...) handler.

Syntax
button("label")("W'H'px")(code)
Example
button("Click me")("80'30'px")(go)
pressed(go) = text("clicked!")

pressed

Defines what runs when the button with the matching code is clicked.

Syntax
pressed(code) = <statement>
Example
button("Pulse")("60'30'px")(p)
pressed(p) = pulse red

<key> pressed

Runs a statement when a single letter key is pressed. Combine with repeat until hold = false for held-key movement.

Syntax
<letter> pressed = <statement>
Example
d pressed = move (bob) 5px right
w pressed = move (bob) 5px up repeat until hold = false
The key must be a single letter (a–z). Handlers do not fire while the Editor pane has keyboard focus — click outside it first.

Stage

pulse

Flashes the entire Stage a color briefly (about 150ms).

Syntax
pulse color
Example
pulse red
pulse #00ffaa

coordinates show

Overlays X and Y pixel labels every 50px along the Stage edges. Helpful while positioning sprites and text.

Syntax
coordinates show
Example
coordinates show
make sprite (circle'green) name = (g) size = ("30'30'px")
teleport (g) to (200'150'px)

Meta

help

Lists every command and its description in the Output pane.

Syntax
help
Example
help
